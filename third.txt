git init - инициализирует новый репозиторий

gitignore - если добавить какой то файл в этот файл,
 то он заморозиться и никакие изменения его не коснуться

git add - подготавливает файлы для добавления в коммит

git commit - подтверждает изменения

git push - отправляет весь репозиторий с файлами на гитхаб
